# Resources

This file is the attribution source of truth for the **Sine-Wave-Estimation** project. Entries are listed in alphabetical order. Every library the project depends on is named below with its author, license, and a source link.

The project itself uses no external datasets or photographs: the signal is a synthetic sine wave generated in code (`np.sin(np.arange(0, 4 * np.pi, step_x))` in `main.py`), so there are no photo or dataset attributions to record. Only the libraries carry attribution requirements.
